Setting up Honeypot in Kali Linux

A way to catch a hackr who is trying to hack my system.

1 / 10
Step 1: I donwloaded a tool from github called pentbox using the command
"git clone https://www.github.com/technicaldada/pentbox" .
2 / 10
Step 2: Now that the donwload was completed, i navigated into the pentbox folder using the command "cd pentbox".
3 / 10
Step 3: The next step was to extract a compressed file named "pentbox.tar.xz" using the following command
"tar -xzvf pentbox.tar.gz" .
4 / 10
Step 4: I then checked the new folder in the directory by using the "ls" command.
Step 5: At this point i navigated to the pentbox-1.8 directory using the command "cd pentbox-1.8 && ls" .
5 / 10
Step 6: I now used the command "./pentbox.rb" to run the ruby tool
6 / 10
Step 7: I then selected the 2nd option(Network Tools)
7 / 10
Step 8: Now it was time to choose the obvious, option 3(Honeypot).
8 / 10
Step 9: Here, i chose option 1(Fast Auto Configuration).
